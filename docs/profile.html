Profile commands

"How do i change my banner, huh?"


/profile me

Usage: /profile me

Description: shows your profile

/profile lookup user_id

/profile lookup user_id [user id]

Description: searches user profiles

/profile customization banner

Usage: /profile customization banner [type] [banner]

Description: changes your profile banner

Arguments:
type - Item ID (can be found in /shop banners) or image URL (e.g https://houl.ml/bobuxcat.png)
banner - Item ID (e.g 5) or image URL (e.g https://houl.ml/bobuxcat.png)

/profile customization color

Usage: /profile customization color [type] [color]

Description: changes your profile color

Arguments:
type - Item ID (can be found in /shop colors) or color HEX-code (e.g #6074d6)
color - Item ID (e.g 5) or color HEX-code (e.g #6074d6)

/profile verification

Description: shows some information about verification
